# 介绍

## 目录说明

- auth 验证部分的函数组件
- comm 全局公用的函数组件
- db 数据库部分的函数组件，和功能组件
- email 邮件发送部分的函数组件和邮件相关的模板
- fs 文件操作部分的函数组件
- http request和response相关的网络通信函数组件，相关功能组件
- pay 支付，订单，购物车以及其相关的函数组件和功能函数
- router 路由转发
